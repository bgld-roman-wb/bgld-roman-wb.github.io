# Burgenland Roman Dictionary

An online dictionary for Burgenland Roman, the Romani dialect spoken by the
Roma community in Burgenland, Austria.

This project is under active development. Source data lives in `data/`
(`data/current/` for the latest dictionary spreadsheet and its structure
spec, `data/archive/` for prior snapshots, `data/background/` for project
context); see [CLAUDE.md](CLAUDE.md) for a summary of the data structure.
